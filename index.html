Connect Four Game

Features

  • Player vs Player mode
  • Two AI difficulty levels (Easy and Hard)
  • Animated piece dropping
  • Game move history
  • Clean, modern UI

Game Preview

Installation

  1. Download the latest release JAR file
  2. Ensure you have Java 17 or later installed
  3. Run the game using: java -jar connectfour.jar
